# foodGPT

**中文名：食物语**

「食物语」是一款基于JavaFX开发的桌面智能饮食顾问应用。它不仅仅是菜谱管理工具，更是一个懂你身体、懂你目标、懂你周期的私人饮食管家，覆盖了"记录→分析→推荐→干预"的完整健康管理闭环。

## 功能特点

### 📊 身体数据管理
- 支持录入身高、体重、年龄等基础身体数据
- 自动计算并展示BMI（身体质量指数）和BMR（基础代谢率）
- 跟踪身体数据变化趋势，生成可视化图表

### 🤖 AI智能食谱推荐
- 基于个人身体数据、健康目标和饮食偏好进行智能推荐
- 支持自定义饮食目标（增肌、减脂、维持体重等）
- 提供多样化的食谱选择，包含详细的食材清单和烹饪步骤

### 🥗 每日营养素追踪
- 实时记录每日饮食摄入，自动计算营养素含量
- 追踪蛋白质、碳水化合物、脂肪三大营养素的摄入量
- 对比每日目标摄入量，清晰展示营养均衡情况

### 🔔 智能干预提醒
- 根据每日营养摄入数据分析，主动推送健康建议
- 当某项营养素摄入不足时，推荐具体的食物补充方案
- 支持定时提醒，养成健康饮食习惯
